后台一键开启樱花、雪花、灯笼、粒子、鼠标跟随、彩带、灰色模式、右键菜单、背景音乐和欢迎弹窗。

== Description ==

九流页面美化特效是一个独立后台控制台插件，不修改主题文件，不修改文章内容。插件使用全站覆盖层实现页面氛围效果，并提供多种兼容注入模式。
